# Team Assignments — BU-MARKET MVP

This document outlines team assignments, roles, and directory ownership for concurrent development of the BU-MARKET platform.

---

## 👥 Development Teams & Scope

### 🔑 Authentication Team
* **Primary Scope**: `accounts/` app. Custom user accounts, verification forms, logins, and signup page.
* **Key Deliverables**: User roles integration, registration pages, custom user validation.

### 🛒 Marketplace Team
* **Primary Scope**: `marketplace/` app. Listing models, categories model, categories sidebar, homepage and browse grids.
* **Key Deliverables**: Browse page, category pages, FTS search logic, listing creation checks.

### 💳 Subscription Team
* **Primary Scope**: `subscriptions/` app. Setup basic, silver, and gold subscription applications, receipt attachments.
* **Key Deliverables**: Listing limits backend rules, pricing plans tables, receipt review.

### 💳 Payments Team
* **Primary Scope**: `payments/` app. Handle flat buyer memberships, payments approvals and logs.
* **Key Deliverables**: Buyer receipt uploads, accountant view.

### 🛡️ Moderation Team
* **Primary Scope**: `moderation/` app. Queue lists for pending products, approvals toggle dashboard.
* **Key Deliverables**: Approved status toggle buttons, moderator dashboard.

### 🎨 Frontend Team
* **Primary Scope**: `templates/`, `static/`. Handle styling, css colors, Bootstrap responsiveness, mobile fixed bottom nav.
* **Key Deliverables**: Mobile layouts, custom stylesheet overrides.

### 🧪 Testing Team
* **Primary Scope**: `tests/` directories in each app. Ensure test suites cover role properties and model limits validations.
* **Key Deliverables**: Unit tests execution, limits enforcement checks.

### 📄 Documentation Team
* **Primary Scope**: `docs/` folder, Markdown guides. Update guides, create daily reports.
* **Key Deliverables**: Installation guide, walkthroughs.
